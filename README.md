# 📌 Project Overview

The **Insurance Premium Prediction API** is a machine learning powered service that estimates health insurance costs based on personal attributes such as age, BMI, smoking status, and number of dependents.

The system exposes a **REST API built with FastAPI** that allows applications to send user data and receive premium predictions in real time.

This project demonstrates an **end-to-end ML workflow**, including:

- Data preprocessing  
- Model training  
- API development  
- Model inference  
- Production-ready backend architecture

# 📡 API Endpoints

## POST `/predict`

Predict insurance premium based on input features.

### Request

```json
{
 "age": 30,
 "sex": "male",
 "bmi": 25.5,
 "children": 2,
 "smoker": "no",
 "region": "southeast"
}

{
 "predicted_premium": 8420.50,

1 Clone the repository

cd insurance-premium-prediction-api
3 Install dependencies
pip install -r requirements.txt
4 Run the API server
